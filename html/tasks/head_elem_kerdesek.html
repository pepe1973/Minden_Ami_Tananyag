HTML - <head> elem - Kérdések

    Kérdések:

  1. Melyik HTML elemnek a leszármazottja és melyiknek a testvére a <head> elem?
    1. <html> és <title>
    2. <html> és <body>
    3. <body> és <html>
    4. Nincsenek ilyen elemek.
    <html> és <body>
  2. Melyik nem metaadato(ka)t tartalmazó elem?
    1. meta
    2. base
    3. href
    4. style
    5. link
    6. script
    7. title
    href
  3. Hol nem jelenik meg alapból a <title> elem tartalma?
    1. A weboldal fülén.
    2. A böngésző címsorában.
    3. A böngészőbeli előzményekben.
    4. A böngésző könyvjelzői között.
    A böngésző könyvjelzői között.
  4. Melyik elemet használjuk a belső stíluslapok tárolására?
    1. <style></style>
    2. <link />
    3. <script></script>
    4. <meta />
    <style></style>
  5. Melyik elemet használjuk a külső stíluslapok tárolására?
    1. <style></style>
    2. <link />
    3. <script></script>
    4. <meta />
    <link />
  6. Melyik nem egy meta elem name="..." tartalma?
    1. keywords
    2. author
    3. cookie
    4. description
    5. viewport
    6. revised
    cookie
