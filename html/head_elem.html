HTML - <head> elem

    Alapvető dolgok:

  1. A <html></html> elemen belül helyezkedik el. A <body></body> elem "testvére".
  2. A <head> elem a következő elemeknek egy közös tárolóhelye, amelyek metaadatokat, azaz adatokat a weboldalhoz tartozó adatokról tárolnak: <title>, <style>, <meta>, <link>, <script>, <base>.
  3. <title></title>:
    • A weboldal címe. Ez jelenik meg az oldal fülén (tab), a böngésző címsorában (title bar), illetve ezen a néven kerül elmentésre a böngésző könyvjelzői, kedvencei közé.
    • Elvárt egy HTML dokumentumban!
    • A KeresőMotor Optimalizálás (Search Engine Optimization - SEO, SEO) aktívan használja.
  4. favicon: az oldal fülén a cím (title) mellett balra található kis kép.
  5. Böngészőkbeli képformátum támogatás.
  6. <style></style>: belső stíluslap (Internal Stylesheet) helye.
  7. <script></script>:
    • JavaScript kód elhelyezése közvetlenül a HTML kódba.
    • Külső JavaScript állományok beimportálása.
  8. <link />: kapcsolat a HTML dokumentum és egy külső erőforrás (resource) között.
    • Leggyakrabban külső stíluslap (External Stylesheet) csatolására.
  9. <meta />: metaadatok, metaadatok a weboldallal kapcsolatban.
    • Reszponzivitáshoz beállítás: viewport
    • Szerző: author
    • Leírás: description
    • Kulcsszavak: keywords
    • Utolsó frissítés: revised
    • Cookie-k beállítása: cookie
    • Frissítés adott időtartam után: refresh
    • Új oldal betöltése adott időtartam után: refresh + url
    • Biztonsági beállítások: content-security-policy
    • Karakterkódolás: charset
    • Http fejléc beállítása: Content-Type
  10. <base />: közös URL forrás a weboldalon megjelenő képekhez és/vagy linkekhez.
    • href: a közös URL forrás címe.
    • target: hol nyíljon meg a hivatkozott oldal.
