HTML - Listák

    Listák - list:

  1. A listák teszik lehetővé, hogy összetartozó dolgokat felsorolásszerűen rögzítsünk.
  2. rendezetlen lista (unordered list): ebben az esetben lényegtelen a dolgok sorrendje.
    • Jelölés.

      <ul><li>Első elem</li><li>Második elem</li></ul>

      tetszőleges számú lista elem (list item) lehet.
    • A lista elemek jelölője alapértelmezetten a teli fekete kör (disc).
    • Lehet még körvonal (circle).
    • Lehet még négyzet (square).
    • Lehet még a semmi (none) is.
    • Jelölés a list-style-type CSS tulajdonsággal történik.

      <ul list-style-type="circle"><li>Első elem</li><li>Második elem</li></ul>

  3. rendezett lista (ordered list): ebben az esetben lényeges a dolgok sorrendje.
    • Jelölés.

      <ol><li>Első elem</li><li>Második elem</li></ol>

      tetszőleges számú lista elem (list item) lehet.
    • A lista elemek jelölője alapértelmezetten az egész számok 1-től kezdve. A type attribútumot használjuk: type="1".
    • Lehet még latin kisbetű: type="a".
    • Lehet még latin nagybetű: type="A".
    • Lehet még római kisbetűs szám: type="i".
    • Lehet még római nagybetűs szám: type="I".
    • A listák sorszámának a kezdetét a start attribútummal tudjuk megadni.
  4. A listák egymásba ágyazhatóak.
  5. leíró lista (description list): ebben az esetben minden dologhoz adunk egy leírást is.
    • Jelölés.

      <dl><dt>Első elem</dt><dd>Első elem leírása</dd></dl>

      tetszőleges számú leírandó elem (description term) és leíró adat (description data) lehet.
    • A lista elemek jelölője alapértelmezetten az egész számok 1-től kezdve.
  6. Az ul, ol, li, dl, dt és dd elemek blokkszintű elemek.
