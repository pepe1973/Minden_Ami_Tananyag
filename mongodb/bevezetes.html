MongoDb - Mongoose - Bevezetés

    Mi az a MongoDb:

  1. MongoDb:
    • ingyenes - free
    • platformfüggetlen - cross-platform
    • nyílt forráskódú - open source
    • NoSQL adatbázis - Non SQL, Non Relational, Non Structured database
  2. Webes felülete: MongoDb
  3. Létezik online és asztali (desktop) verziója is.
  4. Mivel nem RDBMS (Relational Database Management System) nyelv, ezért nincsenek táblák, oszlopok (mező - field), sorok (rekordok).
  5. Nincsen join-tolás sem.
  6. És nincsenek SQL lekérdezések.
  7. Viszont vannak gyűjtemények (collection) és dokumentumok (document).
  8. A dokumentumok javascript, JSON szerű BSON (Binary JSON) objektumokból állnak. Ezek egymásba is ágyazhatóak.

    9. Letöltések, telepítések:

  9. Ha a MongoDb-t a saját gépünkön szeretnénk használni, akkor fel kell rá telepíteni. Ehhez töltsük le a Community Server legutolsó verzióját. Community Server
  10. A szerződés elfogadása után a következő három ablakot jelöljük be. Azután telepíthetünk.
  11. Ennek működtetésre két lehetőségünk van. Feltelepítjük külön a Mongo Shell parancssori ablakot. Mongo Shell
  12. Itt csak egy dolgot kell elfogadni. Ki használhatja a Shell-t.
  13. Vagy ha a Visual Studio Code-ot használjuk, akkor a következő Extension-t telepítsük.
  14. Az Extensions fülön lévő MongoDb levél logóra kattintva indíthatjuk el az alkalmazást.
  15. Egy új Terminal-t nyitva adjuk ki a mongosh utasítást. Ehhez először állítsuk be a Környezeti Változók-nál a következőt a path-nál:

    C:\Program Files\MongoDB\Server\7.0\bin

    Általában ide telepíti a MondoDb-t a Windows.
