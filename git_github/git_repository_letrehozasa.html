Git és GitHub - Git repository létrehozása

    Git adattár (repository):

  1. Lokális adattár (local repository): ez egy rejtett mappa abban a mappában, amiben dolgozunk .git névvel. Ez metaadatokat és az állományok összepréselt adatbázisát (compressed database) tartalmazza.

    HA NEM TUDJUK MIT CSINÁLUNK NE SZERKESSZÜK!

  2. Kétféleképpen hozhatjuk létre.
  3. Első lehetőség: manuálisan hozzuk létre a gépen. Javasolt a Git-tel feltelepülő Git Bash használata.
    • Lépjünk be a mappában ahol dolgozni szeretnénk (F:/git_gyakorlas/proba)
    • Adjuk ki a következő utasítást:

      git init

    • Látható, hogy létrejött a lokális adattár a master ággal (branch).
  4. A létrejövő .git mappa és tartalma, benne a config állomány a lokális beállításokhoz.
  5. Második lehetőség: leklónozunk egy már létező adattárat, például egy GitHub repository-t:

    https://github.com/Projektek/gyakorlas.git

    Ez létrehoz egy gyakorlas nevű mappát és ebbe másol minden szükséges állományt és mappát, azaz a projekt legutolsó verzióját, valamint a .git mappát.

    Ha szeretnénk más néven elmenteni, akkor

    https://github.com/Projektek/gyakorlas.git masNev
