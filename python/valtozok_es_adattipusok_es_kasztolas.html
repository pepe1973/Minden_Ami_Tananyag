Python - Változók, Adattípusok és Kasztolás

    Változók - Variables:

  1. A változók megcímkézett memóriaterületek, amiket az értelmező (interpeter) ideiglenesen lefoglal az adatok tárolására, amíg a program fut.
  2. Mivel a Python típusfüggetlen nyelv, ezért nincs változó deklaráció a nyelvben. Hanem csak simán definiálom, majd használom.

    valtozoNev = valtozoErtek

    Hogy mi lesz a változó adattípusa, majd az interpeter eldönti a szintaktikai szabályai alapján. Például, ha idézőjelek között van a valtozoErtek, akkor szöveg típus (string).
  3. Sőt újradefiniálással bármikor megváltoztathatom a változó adattípusát!

    valtozoNev = "tanuló" - string, majd később

    valtozoNev = 12 - integer.

  4. Megkaphatjuk a változó adattípusát a type() függvény segítségével.
  5. Szabályok a változók azonosítójára (identifier), nevére.
    • Kis-, nagybetű érzékeny (case sensitive): alma ≠Alma
    • Csak betűvel vagy az "_" aláhúzás karakterrel kezdődhet.
    • Nem kezdődhet számmal!
    • Csak alfanumerikus és "_" karaktereket tartalmazhat (a-Z, 0-9, _)!
    • Nem egyezhet meg egyik Python kulcsszóval (keyword) sem.
    • Több szóból álló azonosító esetén javasolt a Camel Case használata. Első szót kisbetűvel kezdjük, majd a többit nagy kezdőbetűvel hozzáfűzzük.

      vasarloNeve, vasarlasIdopontja, vasaroltTermekekListaja

    • Hiba esetén SyntaxError-t kapunk!
  6. Több változónak is adhatunk egyszerre értéket egy sorban, vesszővel elválasztva őket.

    elso, masodik, harmadik = 12, 23, 45

    A hozzárendelés operátora (=) mindkét oldalán ugyanannyi érték kell hogy legyen! Egyébként ValueError-t kapunk!
  7. Egy érték rendelése több változóhoz egy sorban.

    elso = masodik = harmadik = 12

  8. Listák, tuple-ok kicsomagolása (unpacking).

    lista = ["alma", "barack", "körte"]
    x, y, z = lista

    Ekkor x = "alma", y = "barack", z = "körte" lesz.
  9. Változók kiíratása a konzolra a print() függvénnyel történik. Néhány észrevétel.
    • Több változót vesszővel írathatunk ki. Ekkor változók közé szóközt rak a Python a konzolon.
    • Működik minden adattípussal.
    • Több változót +-karakterrel is kiírathatunk. Ekkor változók közé nem rak semmit a Python a konzolon. Célszerű már az értékadásnál elhelyezni a szóközöket!
    • Ez csak string típusú változóknál működik. Egyébként TypeError-t kapunk!
  10. Minden változónak létezik "hatóköre" scope, ahonnan látható. Két fajtája létezik.
  11. globális hatókör (global scope): azoknak a változóknak a hatóköre, amelyek az állományon belül mindenhonnan láthatóak.
  12. lokális hatókör (local scope): azoknak a változóknak a hatóköre, amelyeket egy függvényen, vezérlési szerkezeten vagy osztályon belül definiáltak. Ezeken kívül alapból nem láthatóak. Futtatás esetén NameError-t kapunk.
  13. global: ezzel a kulcsszóval a függvényen belül létrehozott lokális változókat globálissá lehet változtatni.
  14. Globális változó értékét is meg tudjuk így változatani.

    15. Adattípusok:

  15. Az adatok különbözőek lehetnek. Lehetnek szövegek, számok, logikai értékek stb. Ezért különböző műveleteket tudunk rajtuk végrehajtani. Például két számot össze tudunk szorzni, de két szöveget nem.
  16. Ezért a változóknak a különböző adattípusoknak megfelelően kell eltárolni az adatokat. Ezekhez az adattípusokhoz számos, csak ezekkel az adattípusokkal végezhető, művelet tartozik.
  17. Később látni fogjuk, hogy ezek az adattípusok osztályokat határoznak meg, és a műveleteket metódusoknak fogjuk hívni.
  18. Beépített adattípusok a Python-ban.
  19. A Python-ban egy változó akkor kap adattípust, ha hozzárendelünk valamit.
  20. Ugyanez konstruktorokkal.

    21. Literálok - Konstansok a szövegben:

  21. literál: programozás közben számos konstanst elhelyezünk a kódban, főleg az értékadás jobb oldalán. Ezeket nevezzük literáloknak.
  22. Logikai literál kettő van: True és False.

    23. Kasztolás - Típuskényszerítés

  23. Kasztolásról (casting), típukényszerítésről akkor beszélünk, ha egy változó adattípusát explicit módon megváltoztatjuk. Például.

    valtozo = "12", string adattípus

    valtozo = int("12"), integer adattípus

    Tehát egy string típusú adatból csináltunk egy integer típusút.
  24. Ez azért lehetséges, mert a Python objektum-orientált nyelv, és így a típusainak létrehozásához osztályokat használ. Ezek konstruktorainak meg sok minden argumentumot odaadhatunk.
