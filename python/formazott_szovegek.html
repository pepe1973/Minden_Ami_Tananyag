Python - Formázott szövegek

    Szövegek formázott megjelenítése:

  1. Legegyszerűbb formában simán vesszővel, vagy + jellel konkatenálva, összefűzve írjuk a szövegeket.
  2. De ekkor figyelnünk kell a típuskényszerítésre!
  3. Ezért szoktunk inkább formatált sztringeket (formatted string) használni.
  4. Ódivatú formátum (Old Style Format): ekkor a % operátort használjuk. Mivel ez az operátor egy argumentumot tud csak kezelni, ezért több érték kezeléséhez tuple-t használ.
  5. Új stílusú formátum (New Style Format): ekkor a string.format() metódust használjuk.
  6. Sztring interpoláció (String Interpolation): ekkor az f-String-et használunk.
  7. Formázást segítő jelölők (flag).
  8. Néhány példa.
