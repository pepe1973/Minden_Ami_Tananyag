CSS - Háttér

    Háttér - background:

  1. A CSS háttér tulajdonságokkal (property) a weboldalon található HTML elemeknek állíthatjuk be a hátterét. Minden olyan elemnek, amely rendelkezik fizikai méretekkel.
  2. Háttérszín (background-color): beállítása bármely elemre

    selector { background-color: színérték; }

    ahol a színérték a következő szín lehet.
